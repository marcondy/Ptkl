# App Screenshots

Drop your real Hyperia screenshot PNGs here to replace the built-in UI previews.

## Recommended files

| Filename | Screen |
|----------|--------|
| `timeline-dark.png` | Dark mode timeline / peptide stack |
| `timeline-light.png` | Light mode timeline |
| `share-dark.png` | Share plan modal (dark) |
| `share-light.png` | Share plan modal (light) |
| `inventory-dark.png` | Inventory forecast |
| `library-dark.png` | Catalog / library |
| `plans-list-dark.png` | Active plans list |
| `settings-dark.png` | Settings sheet |
| `notifications-dark.png` | Notification time picker |

Then set `useImages: true` in `src/data/screenshots.js`.

Portrait screenshots (~9:19.5) work best. Landscape images are center-cropped to fit phone frames.
